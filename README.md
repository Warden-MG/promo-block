# Promo Block

Небольшой фронтенд-проект на **Vite + SCSS** с адаптивным блоком курсов.

## Требования

- Node.js 18+ (рекомендуется LTS)
- npm 9+

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

После запуска откройте адрес из консоли (обычно `http://localhost:5173`).

## Сборка production-версии

```bash
npm run build
```

Собранные файлы появятся в папке `dist/`.

## Локальный просмотр production-сборки

```bash
npm run preview
```

## Структура проекта

- `index.html` — точка входа страницы
- `src/js/` — JavaScript-модули
- `src/scss/` — стили (SCSS)
- `src/assets/` — шрифты и изображения
